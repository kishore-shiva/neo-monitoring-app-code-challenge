## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start --clear
   <Press "i" to run in iOS>
   <Press "a" to run in Android>
   <Press "w" to open in your default web browser>
   ```
